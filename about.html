ℹ️ About This Project

An interactive educational dashboard designed to bring Data Structures & Algorithms to life through step-by-step visual demonstrations.

🎯 Project Purpose

The DSA Interactive Dashboard was created as a university capstone project for the Data Structures & Algorithms course. Its primary goal is to transform abstract algorithmic concepts into tangible, visual experiences that are accessible to students at all levels of technical proficiency.

Traditional textbooks present algorithms as static pseudocode and diagrams. This dashboard brings them to life — allowing learners to control execution step-by-step, observe data transformations, and develop intuitive understanding through interaction rather than passive reading.

📚 Learning Objectives

Upon completing this interactive module, students should be able to:

🔍 Searching
Compare linear vs binary search performance and explain when each is appropriate based on data characteristics.
⚙️ Sorting
Trace the execution of Bubble, Selection, Insertion, and Merge Sort and analyze their time complexity trade-offs.
📚 Stack
Implement Push/Pop operations and apply the LIFO principle to real problems like expression parsing and undo history.
🔗 Linked List
Describe node-pointer structure, implement insert/delete operations, and contrast with array-based storage.
#️⃣ Hashing
Explain hash functions, predict bucket assignments, and resolve collisions using separate chaining.
🎯 Queue
Apply FIFO semantics to model scheduling systems and implement circular queue behavior.

⚡ Technical Implementation

This project was built using only fundamental web technologies — no frameworks, no external libraries — demonstrating that sophisticated interactive experiences can be achieved with pure HTML, CSS, and JavaScript.
